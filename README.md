# Caffeine Cove - Coffee Shop Website

A modern, fully responsive, and visually engaging coffee shop website built to showcase menus, customer testimonials, and cafe features, complete with an interactive reservation area and location mapping.

 Live Demo & Code
- **Live Website:** [View Live Demo](https://hema7x.github.io/caffeine-cove/)
- **GitHub Repository:** [View Source Code](https://github.com/hema7x/caffeine-cove)

 Key Features
- **Modern & Cozy UI:** Thoughtfully designed layout with a color palette tailored to coffee lovers.
- **Interactive Reservation Area:** A clean, user-friendly table booking form capturing client details, party size, dates, and times.
- **Embedded Location Map:** Fully integrated map guiding customers straight to the cafe.
- **Fully Responsive Layout:** Optimized for a seamless viewing experience across all devices (Mobiles, Tablets, and Desktops).

# Tech Stack
- **HTML5:** Semantic structure and content.
- **CSS3:** Custom properties (variables), modern layouts, and responsive media queries.
- **Font Awesome:** High-quality vector icons for an enhanced visual experience.
